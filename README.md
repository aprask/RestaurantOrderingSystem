### Installing necessary packages:  
* `pip install -r requirements.txt`
### Run the server:
`uvicorn api.main:app --reload`
### Test API by built-in docs:
[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
### Make an .env file
* `DB_HOST=`
* `DB_DATABASE=`
* `DB_PORT=`
* `DB_USER=`
* `DB_PASSWORD=`
* `APP_HOST=`
* `APP_PORT=`
